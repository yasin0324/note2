# 暗黑模式支持

## 启用暗黑模式

```javascript
// tailwind.config.js
module.exports = {
    darkMode: "class", // or 'media'
    // ...
};
```

## 暗黑模式类型

-   媒体查询（media）
-   类名切换（class）
-   系统偏好同步

## 样式定义

-   dark:前缀使用
-   颜色系统适配
-   组件暗色主题

## 切换机制

-   JavaScript 实现
-   状态持久化
-   动画过渡

## 最佳实践

-   配色方案
-   可访问性考虑
-   用户体验优化
