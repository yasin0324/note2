# 安装和配置

## 安装步骤

1. 使用 npm 安装 Tailwind CSS

```bash
npm install -D tailwindcss
npx tailwindcss init
```

2. 配置模板路径

```javascript
// tailwind.config.js
module.exports = {
    content: ["./src/**/*.{html,js}"],
    theme: {
        extend: {},
    },
    plugins: [],
};
```

3. 添加 Tailwind 指令到 CSS

```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```
